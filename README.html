  • Gray circles at the base showing different chroma levels
  • Colored lines radiating from center showing hue directions
  • Point count at bottom (e.g., "Points: 2,185")

Problem: UI shows but NO 3D visualization appears

Solution 1: Check if Three.js loaded
  1. Open debug_check.html in your browser
  2. Look for green ‚úì checkmarks saying "THREE.js loaded successfully"
  3. If you see red ‚úó errors, continue to Solution 2
Solution 2: Check browser console
  1. Press F12 (or Cmd+Option+I on Mac)
  2. Click the "Console" tab
  3. Look for error messages in red

Common errors:

  • THREE is not defined ‚Üí Internet connection issue or CDN blocked
  • CORS error ‚Üí Browser security restriction (see Solution 3)
  • WebGL context lost ‚Üí Graphics driver issue (update drivers)
Solution 3: Run a local web server

Some browsers have strict security when opening HTML files directly. Try:

Option A - Python (if installed):

cd /path/to/munsell_viewer
python -m http.server 8000
# Then open: http://localhost:8000/munsell_visualizer_v2.html

Option B - Node.js (if installed):

npx http-server
# Then open the URL it displays

Option C - VS Code:

Install "Live Server" extension, right-click the HTML file ‚Üí "Open with Live Server"

Problem: Very slow or laggy

Solutions:
  • Lower the "Point Density" slider to "Low"
  • Reduce "Point Size" slider
  • Turn off "Show Boundary Mesh"
  • Close other browser tabs
  • Try a different browser (Chrome/Edge usually have best WebGL performance)

üé® Understanding the Munsell Space

The irregular shape you see represents human color perception:

  • Value (height/Y-axis): Lightness from 0 (black) to 10 (white)
  • Hue (angle): Color around the circle - Red ‚Üí Yellow ‚Üí Green ‚Üí Blue ‚Üí Purple
  • Chroma (distance from center): Saturation/vividness - 0 at center (gray) to 18+ at edge
Key Observations:
  • üü° Yellow reaches the highest chroma (most saturated)
  • üîµ Blue has lower maximum chroma, especially at high values
  • üü£ Purple is very limited at high values (light purples are hard to saturate)
  • The "waist" around Value 5-6 is where most colors reach peak chroma
  • Very dark (Value 0-2) and very light (Value 9-10) colors have low chroma
